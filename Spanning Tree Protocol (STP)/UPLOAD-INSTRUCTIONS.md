
## Steps to Upload

### 1. Download All Files
Download everything from the outputs folder to your computer.

### 2. Create Project Folder
In your local repo at `C:\Users\jonat\OneDrive\Desktop\security-project-portfolio`:

```
security-project-portfolio/
├── SDN-Firewall/
└── Spanning-Tree-Protocol/    ← Create this folder
```

### 3. Copy Files Into Project Folder
Move all the downloaded files into `Spanning-Tree-Protocol/` maintaining the folder structure.

### 4. Add Your Screenshot
- Take a screenshot of your Gradescope results (95/100)
- Save it as `test-results.png`
- Put it in `Spanning-Tree-Protocol/screenshots/`
- Delete the `ADD-SCREENSHOT-HERE.txt` file

### 5. Open GitHub Desktop
1. Open GitHub Desktop
2. It should automatically detect the new files in your repo
3. You should see all the new files listed under "Changes"

### 6. Commit and Push
1. In the commit message box (bottom left), enter:
   ```
   Add Spanning Tree Protocol project
   ```
2. Click "Commit to main"
3. Click "Push origin" (top right)

### 7. Verify on GitHub
Go to https://github.com/JonathanBorda3/security-project-portfolio and make sure the new folder appears with all your files.

## That's It!

Your portfolio should now have both projects:
- SDN-Firewall (completed)
- Spanning-Tree-Protocol (just added)
